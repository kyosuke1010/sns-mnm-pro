# AI SNS Marketing Automation Studio

Threadsを起点にした「AI搭載型SNS運用自動化ツール / AIマーケティングオートメーションツール」のUIプロトタイプです。

## 位置づけ

- ThreadsファーストのSNS投稿生成
- 投稿比較による勝ちパターン判定
- 返信、保存、クリックなどの反応を見込み客化
- 見込み度に応じた自動フォロー分岐
- 特典配布、個別相談、申込までの流れを管理
- フォロワー増加率と保存数の可視化
- バズリサーチ、バズ投稿分析

## 現在できること

- キャンペーンテーマから3つの投稿案を生成
- 投稿プレビュー、コピー、CSV書き出し
- A/B配分の簡易更新
- 配信キューへの追加、削除
- 返信/DM/MAルールのオンオフ
- フォロワー数、1週間前比、アカウント作成時比、保存数の表示
- バズ投稿パターンの表示
- 入力内容をブラウザのlocalStorageに保存

## Canva / ChatGPT連携メモ

ChatGPT側でCanvaを呼び出す場合は、`chatgpt-canva-handoff.md` をコピーして使ってください。
ローカルURLはChatGPT側から直接読めない場合があるため、このファイルにデザイン方針、メニュー構成、画面要件、配色候補をまとめています。

## 次に実装する実運用レイヤー

- 認証、アカウント管理
- 各SNS APIとの接続
- 投稿予約ジョブ
- 反応取得、見込み客DB、見込み度スコアリング
- メール/LINE/CRMなどの自動フォロー連携
- API利用規約とMeta審査に合わせた自動化制御
