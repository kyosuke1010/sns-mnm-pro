# ChatGPT / Canva Handoff Prompt

このファイルは、ChatGPT側でCanvaを呼び出すときにそのまま貼るための指示書です。
ローカルURL `http://127.0.0.1:8790/` はChatGPT側から直接読めない可能性が高いため、下の内容をコピーして使ってください。

---

## ChatGPTに貼る依頼文

以下の内容をもとに、Canvaで「AI搭載型SNS運用自動化ツール / AIマーケティングオートメーションツール」のオリジナルUIデザイン素材を作成してください。

目的は、Threads運用者向けのAI MAツールのブランド素材とUI素材を作ることです。参考元のtthpro風の暗いSaaS感は参考にしますが、背景色・ロゴ・アイコン・カード表現・メニュー順は被りすぎないようにしてください。コピーではなく、独自の業務ツールとして見えるデザインにしてください。

## サービス仮称

第一候補:
`SNS MA Studio`

別案:
- `LeadArc AI`
- `ThreadFlow MA`
- `SyncPost AI`
- `ArcReach Studio`

## サービス概要

Threadsを起点に、投稿生成、予約投稿、保存数分析、フォロワー増加分析、バズリサーチ、バズパターン分析、返信・DM導線、見込み客管理、自動フォロー設定までを一元管理するSNS運用自動化ツールです。

ユーザーに伝えたい印象:
- AIで投稿を作るだけではなく、投稿後の反応を見込み客化する
- 保存数、フォロワー増加、返信、申込までの流れが見える
- 個人事業主、副業者、SNS運用者でも使いやすい
- 高機能だが、メニュー名は分かりやすい

## デザイン方向性

避けたいこと:
- tthproと同じ黒紫の印象に寄せすぎる
- Threads公式ロゴや既存サービスのロゴに似せる
- 紫グラデーションだけの単調な画面にする
- メニュー名を専門用語だらけにする

推奨:
- 全てのベースカラーは、添付写真のようなティールからシアン、ブルー、インディゴへ流れるグラデーションを軸にする
- 暗めのSaaS UIは維持してよいが、黒一色ではなく、グラデーションの上に深い半透明パネルを重ねる
- カード内、入力欄、分析パネル、サイドバー内のベースカラーは濃紺でよい
- メインアクセントはティール、ブルーグリーン、明るすぎないシアンなど
- 紫はAI感の補助色として控えめに使う
- 重要アクションはティール系、分析やAI機能は紫系、注意や課金はアンバー系など役割で色を分ける
- カードは角丸8px前後、業務ツールとして見やすく

## ベースグラデーション指定

添付写真のように、左から右へ `ミント/ティール -> シアン -> ブルー -> インディゴ` に移るグラデーションを全体のベースカラーにしてください。

参考カラー:
- Left / Mint Teal: `#13D6B2`
- Teal Cyan: `#14B9C5`
- Blue: `#2477D4`
- Indigo Blue: `#3E43DC`

推奨CSSイメージ:

```text
linear-gradient(105deg, #13D6B2 0%, #14B9C5 34%, #2477D4 68%, #3E43DC 100%)
```

使い方:
- 背景、ヒーロー、OGP、モバイルUIの上部背景、主要カードの薄い光彩にこのグラデーションを使う
- 全面を明るくしすぎず、UIカード内は濃紺で統一してよい
- カード、入力欄、表、分析パネルの推奨色は `#071426`、`#0B1B2E`、`#0E2236` あたり
- カードにグラデーションを強く入れすぎない。必要なら外枠や上部の薄い光彩だけにする
- ボタンはティールからブルーへの短いグラデーションを基本にする
- 紫はAI機能のワンポイントだけに使い、画面全体を紫ベースにしない

## 画面ごとの要点

ダッシュボード:
- KPIカードを4つ配置
- フォロワー増加率は円グラフ
- 保存数を分かりやすく表示
- 最近の投稿、機能概要、次のアクションを置く

保存・フォロワー分析:
- 現在のフォロワー数
- アカウント作成当初からの増加率
- 1週間前からの増加率
- 保存数
- 見込み客化数
- 円グラフや小さなバーで直感的に表示

バズリサーチ:
- 調べたいテーマ
- 対象期間
- 目的
- 急上昇テーマ
- 投稿へ変換する切り口

バズパターン分析:
- 保存型
- 共感型
- 案内型
- 冒頭、本文、導線に分解
- 自分の投稿テンプレへ変換する印象

AI投稿文生成:
- トピック
- キーワード
- 口調
- 生成数
- 投稿文生成ボタン

プラン・お支払い:
- 初期設定パック
- 月額プラン
- 決済後に有効化される表現
- 料金カードは参考程度で、デザインは独自に

## 出力形式

Canvaで以下を作ってください。

- ロゴ 1案以上
- iPhone縦長ダッシュボードUI 1案
- サイドバーメニューUI 1案
- OGP画像 1案
- 空状態イラスト 1案
- 機能アイコンセット 10個以上

完成後、配色、フォント、余白、アイコンの方向性を簡単に説明してください。

## 重要な注意

参考元と似すぎないようにしてください。特に、背景色、紫の使い方、サイドバーの見た目、ボタンのグラデーション、ロゴ形状はオリジナルにしてください。背景やベース面は、添付写真のようなティールからブルーへのグラデーションを必ず使ってください。

UIは派手なLPではなく、毎日使う業務ツールとして見やすく、スマホでも読みやすいデザインにしてください。
